# Analog Clock

A beautiful, functional analog clock built with HTML, CSS, and JavaScript.

## Features

- Real-time clock updates
- Smooth hand movements
- Clean, modern design
- Responsive layout

## Technologies Used

- HTML5
- CSS3 (Flexbox, Transforms, Transitions)
- JavaScript (Date API, setInterval)

## How to Run

Simply open `index.html` in your web browser.
